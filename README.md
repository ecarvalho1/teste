# Projeto de Versionamento com Git

## O que é Git?

O Git é um sistema de controle de versão que permite acompanhar mudanças em arquivos ao longo do tempo, sendo muito utilizado no desenvolvimento de software.

## O que é GitHub?

O GitHub é uma plataforma online onde podemos armazenar nossos repositórios Git e compartilhar projetos com outras pessoas.
